# Entrega 09 – Requisitos de Infraestrutura

Este documento detalha o ambiente tecnológico necessário para a operação do sistema.

### 9.1. Pilha Tecnológica (Stack)
- **Servidor Web:** Apache 2.4 (Ambiente XAMPP).
- **Banco de Dados (SGBD):** MariaDB/MySQL Versão 10.4 ou superior.
- **Linguagem Backend:** PHP Versão 8.2 ou superior.
- **Linguagem Frontend:** HTML5 (Estrutura semântica conforme pedido).

### 9.2. Configurações de Sistema
- **Sessões:** Gerenciamento via `$_SESSION` do PHP para persistência de login.
- **Ordenação:** Algoritmo de Bubble Sort customizado em PHP para organização de dados em memória.
- **Relações:** Tabelas SQL com integridade referencial (Chaves Estrangeiras) e motor InnoDB.

### 9.3. Dependências de Software
- Navegador moderno (Chrome, Firefox ou Edge).
- Servidor local ativo com o banco de dados `saep_db` devidamente populado.
